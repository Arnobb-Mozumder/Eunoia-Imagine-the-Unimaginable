## 🎬 Animation States

### State 1: Default (Page Load)
```
🔴 PARTICLE SPHERE
Size: Medium radius sphere
Color: Glowing indigo particles
Motion: Gentle, slow rotation
State: At rest, waiting for interaction
```

### State 2: Scrolling Down
```
🌊 PARTICLE MORPHING
Phase 1 (0-25%):    Sphere → Wavy Blob
Phase 2 (25-50%):   Blob → Undulating Wave
Phase 3 (50-75%):   Wave → Stretched Torus
Phase 4 (75-100%):  Torus → Spiral
Motion: Smooth interpolation between shapes
Speed: Responsive to scroll speed
```

### State 3: Click on Screen
```
💥 PARTICLE EXPLOSION
Trigger: Mouse click anywhere
Effect: All particles blast outward in random directions
Duration: ~2 seconds
Physics: Exponential decay with damping
Recovery: Particles regroup back to sphere
Animation: Smooth regrouping motion
```

### State 4: Mouse Movement
```
🎯 PARTICLE ATTRACTION
Effect: Particles subtly follow cursor
Intensity: Subtle, not distracting
Range: Affects visible particle movement
Response: Smooth, continuous tracking
Result: Creates responsive, engaging feel
```

## 🌈 Visual Effects

### Particle Glow
```
Particle center: Bright, full color
Particle edge:   Fades to transparent
Multiple overlapping: Creates bright clusters
Additive blend:  Colors combine (additive, not subtractive)
```

### Panel Effect (Glassmorphism)
```
Background:      Semi-transparent dark blue (75% opacity)
Backdrop blur:   20px frosted glass effect
Border:          Thin light border for definition
Depth:           Creates layered, sophisticated look
```

### Color Gradient in Particles
```
Brightest:  Center of particles (#6366f1 + full glow)
Glow zone:  Around particles (cyan/purple halo)
Shadow:     Slight depth from lighting
```

## 🎨 Browser Rendering

The browser will render in this order:
1. **Layer 1:** Black background
2. **Layer 2:** WebGL canvas with particle animation
3. **Layer 3:** Glass panel with text and content
4. **Layer 4:** Navigation bar on top

Result: Particles visible behind content, nothing blocks readability.
